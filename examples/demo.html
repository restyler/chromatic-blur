Chromatic Aberration Effect

This navigation bar uses SVG filters to create a stunning chromatic aberration effect. The background text is split into RGB channels and offset in different directions, creating the rainbow-like reflections you see through the glass navigation bar.

How It Works

The effect combines multiple techniques: color channel splitting (red offset right, blue offset left), turbulence for organic distortion, displacement mapping for wavy effects, and Gaussian blur for that soft lens blur look. Scroll up and down to see how the text behind interacts with the glass effect.

Multiple Reflections

The layered approach creates multiple "reflections" of the background text, with each color channel creating its own ghosted copy. This mimics the way light refracts through glass or camera lenses with chromatic aberration.

Keep Scrolling

Notice how the navigation bar stays fixed at the top while the background text scrolls behind it, creating a dynamic interaction between the content and the blur effect. The color aberration becomes especially visible where text passes behind the navigation.
